Анализатор тональности отзывов на русском языке
Python
License
Model
Простая консольная программа, которая определяет, является ли русскоязычный отзыв позитивным, нейтральным или негативным, используя предобученную модель blanchefort/rubert-base-cased-sentiment-rusentiment.
Возможности

Мгновенная классификация текста
Поддержка длинных отзывов (до ~400 слов)
Работает полностью офлайн после первой загрузки модели
Не требует GPU (работает на CPU)

Содержимое репозитория:
├── AImodel.py      ← загрузка модели и функция предсказания
├── start.py        ← основной интерфейс (цикл ввода-вывода)
└── README.md       ← эта инструкция

Установка и запуск:
1. Клонируйте или скачайте проект
git clone https://github.com/ваш-username/sentiment-analyzer-ru.git
cd sentiment-analyzer-ru

2. (Рекомендуется) Создайте виртуальное окружение:
python -m venv venv
source venv/bin/activate      # Linux / macOS
venv\Scripts\activate         # Windows

3. Установите зависимости:
pip install torch transformers

Если у вас нет видеокарты NVIDIA или возникают проблемы с установкой torch:
pip install torch --index-url https://download.pytorch.org/whl/cpu
pip install transformers

4. Запустите программу:
python start.py

При первом запуске автоматически загрузится модель (~500 МБ). Последующие запуски будут мгновенными.

Как пользоваться:
Введите текст отзыва: Отличный товар, рекомендую всем!
Характеристика отзыва: позитивный

Хотите проанализировать еще отзыв? (да/нет): да

Введите текст отзыва: Нормально, но могло быть и лучше.
Характеристика отзыва: нейтральный

Хотите проанализировать еще отзыв? (да/нет): нет

Программа принимает ответы «да»/«нет» в любом регистре.

Примеры результата
Ввод,Результат
"Обожаю этот магазин, всё супер!"
"Ничего особенного, обычный товар"
"Ужасный сервис, больше никогда"

Результат
позитивный
нейтральный
негативный

Технические детали:
Модель: blanchefort/rubert-base-cased-sentiment-rusentiment
Классы: 0 → нейтральный, 1 → позитивный, 2 → негативный
Максимальная длина текста: 512 токенов (автоматически обрезается)
Язык: только русский (на других языках точность сильно падает)
